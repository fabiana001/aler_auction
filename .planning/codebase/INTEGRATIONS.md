# External Integrations

**Analysis Date:** 2026-04-14

## APIs & External Services

**Geocoding:**
- Google Maps Geocoding API - Converts property address strings to lat/lng coordinates
  - SDK/Client: `googlemaps` (`googlemaps.Client`)
  - Auth: `GOOGLE_MAPS_API_KEY` environment variable (loaded via `python-dotenv`)
  - Implementation: `src/aler_auctions/data_integration/geocoder.py`
  - Caching: Results cached to `data/geocoding_cache.json` to minimize API calls; cache persisted incrementally every 20 new geocodes

**Wayback Machine (Internet Archive):**
- CDX Search API (`https://web.archive.org/cdx/search/cdx`) - Discovers archived snapshots of the ALER auction website
- Wayback Page Fetcher (`https://web.archive.org/web/{timestamp}/{url}`) - Downloads archived HTML pages
  - SDK/Client: raw `requests.Session` (no dedicated SDK)
  - Auth: None (public API)
  - Implementation: `src/aler_auctions/data_extraction/wayback_client.py`
  - Rate limiting: configurable `delay_seconds` (default 1s) between page downloads

**ALER Milano Website (Direct Scraping):**
- `HistoricalAuctionClient` scrapes the live ALER website for historical auction result PDFs
  - SDK/Client: `requests.Session` with browser-like `User-Agent` header
  - Auth: None (public website)
  - Implementation: `src/aler_auctions/data_extraction/historical_client.py`
  - Output: downloads PDF files to a local `output_dir`

## Data Storage

**Databases:**
- None - No database used; all data is stored as local files

**File Storage:**
- Local filesystem only
  - Raw HTML snapshots: saved to configurable output directory as `{timestamp}.html`
  - Downloaded PDFs: saved to configurable output directory
  - Geocoding cache: `data/geocoding_cache.json` (JSON, persisted between runs)
  - Processed datasets: CSV/Excel files via pandas (`openpyxl` for `.xlsx`)
  - Map output: `map.html` at project root (generated by `gmplot`)

**Caching:**
- File-based JSON cache for geocoding results (`data/geocoding_cache.json`)
- File-based deduplication for already-downloaded HTML snapshots and PDFs (checked with `Path.exists()` before re-downloading)

## Authentication & Identity
